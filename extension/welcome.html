Web Annotator for Pi

Turn page feedback into Pi tasks

Mark an element or a passage in Firefox, describe the change, then copy it to Pi or send it directly to your active Pi session.

1. Start annotating

  1. Open a normal web page, then select Web Annotator for Pi from the Firefox extensions menu. Pin it to the toolbar for one-click access.
  2. Select the toolbar icon or press Alt+Shift+A.
  3. Click an element, or switch to Text and select a passage.
  4. Write the change you want and select Save.

Firefox blocks extensions on protected pages such as about:, the Add-ons site, PDFs, and view-source: pages.

2. Connect Pi

If Pi is not installed, follow the instructions at pi.dev. Review the Web Annotator for Pi package source, then install it:

pi install npm:pi-web-annotator

Restart Pi and start the local annotation bridge:

/annotation-server start

3. Send feedback

Use Save and send for a new annotation, the arrow beside one saved annotation, or Send to Pi for every pending annotation.

The first send opens a separate consent page. Firefox will show exactly which page details can be shared. Nothing is sent until you approve that access and send again.
